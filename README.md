## StockListener 介绍
iOS开源股票软件 (声音提示，k线，筹码集中度，画线)

开源软件，使用了新浪和腾讯的股票接口，仅供学习使用

## 功能

* 时时股票数据
* 时时K线，KDJ，成交量，买卖10档
* 声音提示，开机页面显示股票信息
* 画线功能
* 成本计算
* 筹码分布
* 筹码集中趋势
* 涨跌趋势
